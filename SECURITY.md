# Security Policy

## Supported Version
Current maintained line:
- `1.0.x`

## Reporting a Vulnerability
Please do not open public issues for security reports.

Send details to: `giridharuppili2004@gmail.com`

Include:
- Impact summary
- Steps to reproduce
- Affected component(s)
- Suggested mitigation (if known)

## Response Targets
- Acknowledge report within 48 hours
- Share remediation plan after triage
- Coordinate disclosure after fix

## Scope Notes
This is a student prototype. Production hardening, formal penetration testing, and SLA-backed support are not yet in place.
