# Contributing to REC LostLink

Thank you for contributing to the REC LostLink ecosystem. Your help improves the experience for all students at Rajalakshmi Engineering College.

---

## How to Help

* **Report Bugs:** Open a GitHub Issue with clear steps to reproduce and screenshots.
* **Suggest Features:** Open an issue and apply the "enhancement" label.
* **Pull Requests:** Fork the repository, create a branch from "main," and submit your PR once standards are met.

---

## Technical Standards

### Backend (FastAPI)
* Follow PEP 8 guidelines.
* Use type hints for all function arguments and return types.

### Frontend (React & React Native)
* Use functional components and hooks.
* Ensure responsive design for both the mobile app and admin dashboard.

### Development Workflow
* **Branches:** Use kebab-case (e.g., feature/add-notifications).
* **Commits:** Use descriptive messages (e.g., fix: resolve login loop).
* **Documentation:** Update LOGIC.md for any logic changes and README.md for setup changes.
